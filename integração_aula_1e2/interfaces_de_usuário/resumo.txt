A interface de usuário, também conhecida como interface gráfica de usuário (GUI), é uma interface mais simples e menos complexa de se usar, se comparada com a interface de linha de comando. É um prorama evocado pela linha de comando onde, todas as ações executadas na interface gráfica, são executadas em paralelo (segundo plano) pela linhade comando, ou seja, é um programa que apresenta entradas diferentes para se realizar os mesmos comandos que a CLI.
