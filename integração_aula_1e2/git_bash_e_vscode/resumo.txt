O git bash é um programa responsável por interpretar as linhas de comando da interface do linux e transcrevê-las para o windows. Sua interface de linha de comando (CLI) é semelhante ao terminal do Linux. São interfaces que necessitam de grande conhecimento de linhas de comando. O VS Code é um software de programção onde se é possível rodar interfaces de linhas de comando diferentes e importar bibliotecas de códigos.
